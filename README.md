# Website Repository

## GitHub Pages Setup

Untuk mengaktifkan GitHub Pages:

1. Buka repository: https://github.com/gimnasirwandi-boop/Website
2. Klik **Settings** (di menu atas repository)
3. Scroll ke bagian **Pages** (di sidebar kiri)
4. Di bagian **Source**:
   - Pilih **"Deploy from a branch"**
   - **Branch**: pilih `main`
   - **Folder**: pilih `/ (root)`
5. Klik **Save**

Setelah diaktifkan, tunggu 1-5 menit untuk GitHub Pages build dan deploy.

## URL Setelah Aktif
